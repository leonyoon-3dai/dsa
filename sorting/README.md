# Sorting Algorithm Practice (정렬 알고리즘 연습)

Python으로 작성된 정렬 알고리즘 예제 10개. 각 파일에 한국어 라인별 주석 포함.

## Examples

| # | File | Topic |
|---|------|-------|
| 1 | `01_bubble_sort.py` | Bubble Sort (버블 정렬) |
| 2 | `02_selection_sort.py` | Selection Sort (선택 정렬) |
| 3 | `03_insertion_sort.py` | Insertion Sort (삽입 정렬) |
| 4 | `04_merge_sort.py` | Merge Sort (병합 정렬) |
| 5 | `05_quick_sort.py` | Quick Sort (퀵 정렬) |
| 6 | `06_heap_sort.py` | Heap Sort (힙 정렬) |
| 7 | `07_counting_sort.py` | Counting Sort (계수 정렬) |
| 8 | `08_radix_sort.py` | Radix Sort (기수 정렬) |
| 9 | `09_shell_sort.py` | Shell Sort (셸 정렬) |
| 10 | `10_tim_sort.py` | Tim Sort (팀 정렬) |

## Usage

```bash
python3 01_bubble_sort.py
```
