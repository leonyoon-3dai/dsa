# 01. Bubble Sort (버블 정렬)

## 문제 정의

주어진 배열을 오름차순으로 정렬하라.

```
인접한 두 원소를 비교하여 순서가 잘못되었으면 교환한다.
이 과정을 배열이 정렬될 때까지 반복한다.
```

이름의 유래: 가장 큰 원소가 거품(bubble)처럼 배열의 끝으로 떠오른다.

---

## 접근법 1: 기본 버블 정렬

매 패스마다 인접한 원소를 비교·교환합니다.

```python
def bubble_sort_basic(arr: List[int]) -> List[int]:
```

### 라인별 설명

```python
result = arr[:]
n = len(result)
```
- 원본 배열을 보호하기 위해 복사본을 생성합니다.
- 배열 길이를 `n`에 저장합니다.

```python
for i in range(n - 1):
    for j in range(n - 1 - i):
        if result[j] > result[j + 1]:
            result[j], result[j + 1] = result[j + 1], result[j]
```
- 외부 루프: `n-1`번의 패스를 수행합니다. 매 패스마다 가장 큰 미정렬 원소가 제자리로 이동합니다.
- 내부 루프: `n-1-i`번 비교합니다. 뒤쪽 `i`개는 이미 정렬되었으므로 제외합니다.
- `result[j] > result[j+1]`이면 두 원소를 교환합니다.

### 실행 예제: arr = [64, 34, 25, 12]

```
패스 1:
  j=0: [64, 34, 25, 12] → 64>34 → [34, 64, 25, 12]
  j=1: [34, 64, 25, 12] → 64>25 → [34, 25, 64, 12]
  j=2: [34, 25, 64, 12] → 64>12 → [34, 25, 12, 64]  ← 64 확정

패스 2:
  j=0: [34, 25, 12, 64] → 34>25 → [25, 34, 12, 64]
  j=1: [25, 34, 12, 64] → 34>12 → [25, 12, 34, 64]  ← 34 확정

패스 3:
  j=0: [25, 12, 34, 64] → 25>12 → [12, 25, 34, 64]  ← 25 확정

결과: [12, 25, 34, 64]
```

## 접근법 2: 최적화된 버블 정렬 (조기 종료)

교환이 없으면 이미 정렬 완료로 판단하여 조기 종료합니다.

```python
def bubble_sort_optimized(arr: List[int]) -> List[int]:
```

### 라인별 설명

```python
for i in range(n - 1):
    swapped = False
    for j in range(n - 1 - i):
        if result[j] > result[j + 1]:
            result[j], result[j + 1] = result[j + 1], result[j]
            swapped = True
    if not swapped:
        break
```
- `swapped` 플래그를 사용하여 이번 패스에서 교환이 발생했는지 추적합니다.
- 한 패스에서 교환이 한 번도 없었다면 배열이 이미 정렬된 것이므로 `break`합니다.
- **최선의 경우**: 이미 정렬된 배열에서는 1번의 패스만 수행하여 O(n)에 종료합니다.

### 실행 예제: arr = [1, 2, 3, 4, 5] (이미 정렬됨)

```
패스 1:
  j=0: 1<2 → 교환 없음
  j=1: 2<3 → 교환 없음
  j=2: 3<4 → 교환 없음
  j=3: 4<5 → 교환 없음
  swapped = False → break!

결과: 1번의 패스만으로 종료 (O(n))
```

## 접근법 3: 재귀 버블 정렬

재귀적으로 한 패스씩 수행합니다.

```python
def bubble_sort_recursive(arr: List[int], n: int = None) -> List[int]:
```

### 라인별 설명

```python
if n is None:
    arr = arr[:]
    n = len(arr)
```
- 최초 호출 시 복사본을 생성하고 `n`을 배열 길이로 초기화합니다.

```python
if n == 1:
    return arr
```
- **Base Case**: 배열 크기가 1이면 정렬이 완료된 것이므로 반환합니다.

```python
for i in range(n - 1):
    if arr[i] > arr[i + 1]:
        arr[i], arr[i + 1] = arr[i + 1], arr[i]
return bubble_sort_recursive(arr, n - 1)
```
- 한 번의 패스로 가장 큰 원소를 맨 뒤로 보냅니다.
- `n-1`로 재귀 호출하여 나머지 부분을 정렬합니다.

### 실행 예제: arr = [3, 1, 2]

```
호출 1 (n=3): [3, 1, 2] → 패스 → [1, 2, 3] → 재귀(n=2)
호출 2 (n=2): [1, 2, 3] → 패스 → [1, 2, 3] → 재귀(n=1)
호출 3 (n=1): Base case → 반환

결과: [1, 2, 3]
```
