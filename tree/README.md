# Tree Algorithm Practice (트리 알고리즘 연습)

Python으로 작성된 트리 알고리즘 예제 10개. 각 파일에 한국어 라인별 주석 포함.

## Examples

| # | File | Topic |
|---|------|-------|
| 1 | `01_inorder_traversal.py` | Inorder Traversal (중위 순회) |
| 2 | `02_preorder_postorder.py` | Preorder & Postorder (전위/후위 순회) |
| 3 | `03_level_order_traversal.py` | Level Order Traversal (레벨 순회) |
| 4 | `04_max_depth.py` | Maximum Depth (최대 깊이) |
| 5 | `05_validate_bst.py` | Validate BST (이진 탐색 트리 검증) |
| 6 | `06_lowest_common_ancestor.py` | LCA (최소 공통 조상) |
| 7 | `07_binary_search_tree.py` | BST Insert/Search/Delete (이진 탐색 트리 연산) |
| 8 | `08_diameter_of_tree.py` | Diameter of Binary Tree (이진 트리 지름) |
| 9 | `09_serialize_deserialize.py` | Serialize & Deserialize (직렬화/역직렬화) |
| 10 | `10_balanced_binary_tree.py` | Balanced Binary Tree (균형 이진 트리) |

## Usage

```bash
python3 01_inorder_traversal.py
```
