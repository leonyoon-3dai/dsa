# Graph Algorithm Practice (그래프 알고리즘 연습)

Python으로 작성된 그래프 알고리즘 예제 10개. 각 파일에 한국어 라인별 주석 포함.

## Examples

| # | File | Topic |
|---|------|-------|
| 1 | `01_bfs.py` | BFS (너비 우선 탐색) |
| 2 | `02_dfs.py` | DFS (깊이 우선 탐색) |
| 3 | `03_dijkstra.py` | Dijkstra (다익스트라 최단 경로) |
| 4 | `04_number_of_islands.py` | Number of Islands (섬의 개수) |
| 5 | `05_topological_sort.py` | Topological Sort (위상 정렬) |
| 6 | `06_cycle_detection.py` | Cycle Detection (순환 탐지) |
| 7 | `07_bellman_ford.py` | Bellman-Ford (벨만-포드) |
| 8 | `08_floyd_warshall.py` | Floyd-Warshall (플로이드-워셜) |
| 9 | `09_kruskal_mst.py` | Kruskal's MST (최소 신장 트리) |
| 10 | `10_bipartite_check.py` | Bipartite Check (이분 그래프 판별) |

## Usage

```bash
python3 01_bfs.py
```
