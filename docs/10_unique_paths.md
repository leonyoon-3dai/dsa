# 10. Unique Paths (고유 경로)

## 문제 정의

`m x n` 격자의 **왼쪽 상단(0,0)**에서 **오른쪽 하단(m-1, n-1)**까지 이동하는 경로의 수를 구하라. 이동은 **오른쪽** 또는 **아래쪽**으로만 가능하다.

- LeetCode #62

---

## 핵심 아이디어

`dp[i][j]` = (i, j) 위치에 도달하는 경로의 수

(i, j)에 도달하려면:
- **위에서** 내려오거나: `dp[i-1][j]`
- **왼쪽에서** 오거나: `dp[i][j-1]`

**점화식**: `dp[i][j] = dp[i-1][j] + dp[i][j-1]`

---

## 코드 라인별 설명

```python
dp = [[1] * n for _ in range(m)]
```
- `m x n` 크기의 2D 배열을 **1로** 초기화합니다.
- 첫 번째 행: 오른쪽으로만 갈 수 있으므로 모두 1
- 첫 번째 열: 아래로만 갈 수 있으므로 모두 1

```python
for i in range(1, m):
    for j in range(1, n):
```
- (1,1)부터 (m-1, n-1)까지 순회합니다.
- 0번째 행/열은 이미 1로 초기화되어 있습니다.

```python
        dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
```
- **위에서 오는 경로 수** + **왼쪽에서 오는 경로 수**
- 두 방향에서 오는 경로를 합산합니다.

```python
return dp[m - 1][n - 1]
```
- 우하단 도달 경로 수를 반환합니다.

---

## 실행 예제 1: 3x3 격자

```
  j: 0  1  2
i:
0   [1] [1] [1]
1   [1] [2] [3]
2   [1] [3] [6]

dp[1][1] = dp[0][1] + dp[1][0] = 1 + 1 = 2
dp[1][2] = dp[0][2] + dp[1][1] = 1 + 2 = 3
dp[2][1] = dp[1][1] + dp[2][0] = 2 + 1 = 3
dp[2][2] = dp[1][2] + dp[2][1] = 3 + 3 = 6

결과: 6가지 경로
```

**6가지 경로 시각화:**
```
1. →→↓↓    2. →↓→↓    3. →↓↓→
4. ↓→→↓    5. ↓→↓→    6. ↓↓→→
```

## 복잡도

| 항목 | 값 |
|------|-----|

| 공간 복잡도 | O(m x n), 1D 최적화 시 O(n) |

> 참고: 수학적으로 조합 공식 C(m+n-2, m-1)로도 풀 수 있습니다.
