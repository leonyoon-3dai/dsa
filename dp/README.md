# Dynamic Programming Practice (동적 프로그래밍 연습)

Python으로 작성된 DP 예제 10개. 각 파일에 한국어 라인별 주석 포함.

## Examples

| # | File | Topic |
|---|------|-------|
| 1 | `01_fibonacci.py` | Fibonacci (피보나치) |
| 2 | `02_climbing_stairs.py` | Climbing Stairs (계단 오르기) |
| 3 | `03_coin_change.py` | Coin Change (동전 거스름돈) |
| 4 | `04_longest_common_subsequence.py` | LCS (최장 공통 부분 수열) |
| 5 | `05_knapsack.py` | 0/1 Knapsack (배낭 문제) |
| 6 | `06_longest_increasing_subsequence.py` | LIS (최장 증가 부분 수열) |
| 7 | `07_edit_distance.py` | Edit Distance (편집 거리) |
| 8 | `08_house_robber.py` | House Robber (도둑 문제) |
| 9 | `09_maximum_subarray.py` | Maximum Subarray (최대 부분 배열) |
| 10 | `10_unique_paths.py` | Unique Paths (고유 경로) |

## Usage

```bash
python3 01_fibonacci.py
```
