# 08. House Robber (도둑 문제)

## 문제 정의

일렬로 늘어선 집들에 각각 돈이 있다. **인접한 두 집을 연속으로 털 수 없을 때**, 훔칠 수 있는 **최대 금액**을 구하라.

- LeetCode #198

---

## 핵심 아이디어

`dp[i]` = i번째 집까지 고려했을 때 훔칠 수 있는 최대 금액

각 집에서의 선택:
1. **i번째 집을 턴다**: dp[i-2] + nums[i] (i-1번째는 건너뜀)
2. **i번째 집을 안 턴다**: dp[i-1]

**점화식**: `dp[i] = max(dp[i-1], dp[i-2] + nums[i])`

---

## 코드 라인별 설명

```python
if not nums:
    return 0
```
- 집이 없으면 0을 반환합니다.

```python
if len(nums) == 1:
    return nums[0]
```
- 집이 1개면 그 집의 금액을 반환합니다.

```python
n = len(nums)
dp = [0] * n
```
- dp 배열을 0으로 초기화합니다.

```python
dp[0] = nums[0]
```
- 첫 번째 집만 있을 때: 그 집을 털면 됩니다.

```python
dp[1] = max(nums[0], nums[1])
```
- 두 번째 집까지 있을 때: 둘 중 금액이 큰 집을 선택합니다.
- 두 집은 인접하므로 하나만 선택 가능합니다.

```python
for i in range(2, n):
    dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])
```
- **핵심 점화식**:
  - `dp[i-1]`: i번째 집을 **안 터는** 경우 (이전까지의 최대)
  - `dp[i-2] + nums[i]`: i번째 집을 **터는** 경우
    - i-1번째는 건너뛰고, i-2까지의 최대 + 현재 집의 금액
  - 둘 중 큰 값을 선택합니다.

```python
return dp[n - 1]
```
- 마지막 집까지 고려한 최대 금액을 반환합니다.

---

## 실행 예제 1: nums = [2, 7, 9, 3, 1]

```
집:     [2]  [7]  [9]  [3]  [1]
인덱스:  0    1    2    3    4

dp[0] = 2
dp[1] = max(2, 7) = 7
dp[2] = max(dp[1], dp[0]+9) = max(7, 2+9) = max(7, 11) = 11
dp[3] = max(dp[2], dp[1]+3) = max(11, 7+3) = max(11, 10) = 11
dp[4] = max(dp[3], dp[2]+1) = max(11, 11+1) = 12

결과: 12 (집 0 + 집 2 + 집 4 = 2 + 9 + 1)
```

## 실행 예제 2: nums = [1, 2, 3, 1]

```
dp[0] = 1
dp[1] = max(1, 2) = 2
dp[2] = max(2, 1+3) = max(2, 4) = 4
dp[3] = max(4, 2+1) = max(4, 3) = 4

결과: 4 (집 0 + 집 2 = 1 + 3)
```

## 실행 예제 3: nums = [4, 1, 2, 7, 5, 3, 1]

```
dp[0] = 4
dp[1] = max(4, 1) = 4
dp[2] = max(4, 4+2) = 6
dp[3] = max(6, 4+7) = 11
dp[4] = max(11, 6+5) = 11
dp[5] = max(11, 11+3) = 14
dp[6] = max(14, 11+1) = 14

결과: 14 (집 0 + 집 3 + 집 5 = 4 + 7 + 3)
```
