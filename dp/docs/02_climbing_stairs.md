# 02. Climbing Stairs (계단 오르기)

## 문제 정의

계단이 n개 있다. 한 번에 **1칸** 또는 **2칸** 오를 수 있을 때, 꼭대기에 도달하는 **방법의 수**를 구하라.

- LeetCode #70

---

## 핵심 아이디어

i번째 계단에 도달하려면:
- **(i-1)번째 계단에서 1칸** 오르거나
- **(i-2)번째 계단에서 2칸** 오르거나

따라서: `dp[i] = dp[i-1] + dp[i-2]` (피보나치와 동일한 구조!)

---

## 코드 라인별 설명

```python
def climbing_stairs(n: int) -> int:
```
- 계단 수 `n`을 입력받아 방법의 수를 반환합니다.

```python
if n <= 1:
    return 1
```
- **Base Case**: 계단이 0개(출발점) 또는 1개이면 방법은 1가지뿐입니다.

```python
dp = [0] * (n + 1)
```
- DP 테이블 생성: `dp[i]` = i번째 계단에 도달하는 방법의 수

```python
dp[0] = 1
dp[1] = 1
```
- 초기값: 출발점(0번째)에 도달하는 방법 = 1가지, 1번째 계단 = 1가지

```python
for i in range(2, n + 1):
    dp[i] = dp[i - 1] + dp[i - 2]
```
- **점화식**: i번째 계단 = (i-1)에서 1칸 + (i-2)에서 2칸
- 2번째 계단부터 n번째 계단까지 순차적으로 계산

```python
return dp[n]
```
- n번째 계단에 도달하는 방법의 수를 반환

---

## 실행 예제 1: n = 4

```
계단: [0] [1] [2] [3] [4]

dp[0] = 1  (출발점)
dp[1] = 1  (1칸 오르기)
dp[2] = dp[1] + dp[0] = 1 + 1 = 2
dp[3] = dp[2] + dp[1] = 2 + 1 = 3
dp[4] = dp[3] + dp[2] = 3 + 2 = 5

결과: 5가지 방법
```

**5가지 경로:**
1. 1-1-1-1
2. 1-1-2
3. 1-2-1
4. 2-1-1
5. 2-2

## 실행 예제 2: n = 3

```
dp[0] = 1
dp[1] = 1
dp[2] = 1 + 1 = 2
dp[3] = 2 + 1 = 3

결과: 3가지 방법
```

**3가지 경로**: 1-1-1, 1-2, 2-1

## 실행 예제 3: n = 1~10

```
n=1:  1가지
n=2:  2가지
n=3:  3가지
n=4:  5가지
n=5:  8가지
n=6:  13가지
n=7:  21가지
n=8:  34가지
n=9:  55가지

```
