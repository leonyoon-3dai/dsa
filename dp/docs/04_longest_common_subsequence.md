# 04. Longest Common Subsequence (최장 공통 부분 수열)

## 문제 정의

두 문자열 `text1`, `text2`가 주어질 때, **가장 긴 공통 부분 수열(LCS)**의 길이를 구하라.

- 부분 수열: 원래 순서를 유지하면서 일부 문자를 선택한 것 (연속일 필요 없음)
- LeetCode #1143

**예시**: `"abcde"`와 `"ace"` → LCS = `"ace"` (길이 3)

---

## 핵심 아이디어

2D DP 테이블을 사용합니다:
- `dp[i][j]` = `text1[:i]`와 `text2[:j]`의 LCS 길이
- 두 문자가 같으면: 대각선 값 + 1
- 다르면: 위 또는 왼쪽 중 큰 값

---

## 코드 라인별 설명

```python
m = len(text1)
n = len(text2)
```
- 두 문자열의 길이를 저장합니다.

```python
dp = [[0] * (n + 1) for _ in range(m + 1)]
```
- `(m+1) x (n+1)` 크기의 2D 배열을 0으로 초기화합니다.
- 0번째 행/열은 빈 문자열과의 LCS = 0을 의미합니다.

```python
for i in range(1, m + 1):
    for j in range(1, n + 1):
```
- text1의 각 문자(i)와 text2의 각 문자(j)를 비교합니다.

```python
        if text1[i - 1] == text2[j - 1]:
            dp[i][j] = dp[i - 1][j - 1] + 1
```
- **두 문자가 같으면**: 이전 LCS에 1을 더합니다.
- `dp[i-1][j-1]`: 양쪽 문자열에서 한 글자씩 뺀 상태의 LCS
- 같은 문자를 찾았으므로 +1

```python
        else:
            dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
```
- **두 문자가 다르면**: 둘 중 하나를 건너뛰는 경우의 최대값
- `dp[i-1][j]`: text1의 현재 문자를 건너뛴 경우
- `dp[i][j-1]`: text2의 현재 문자를 건너뛴 경우

```python
return dp[m][n]
```
- 전체 두 문자열의 LCS 길이를 반환합니다.

---

## 실행 예제 1: text1 = "abcde", text2 = "ace"

```
    ""  a  c  e
""   0  0  0  0
a    0  1  1  1
b    0  1  1  1
c    0  1  2  2
d    0  1  2  2
e    0  1  2  3

dp[5][3] = 3 → LCS = "ace"
```

**과정 상세:**
- dp[1][1]: 'a' == 'a' → dp[0][0] + 1 = 1
- dp[2][1]: 'b' != 'a' → max(dp[1][1], dp[2][0]) = max(1, 0) = 1
- dp[3][2]: 'c' == 'c' → dp[2][1] + 1 = 2
- dp[5][3]: 'e' == 'e' → dp[4][2] + 1 = 3

## 실행 예제 2: text1 = "AGGTAB", text2 = "GXTXAYB"

```
      ""  G  X  T  X  A  Y  B
""     0  0  0  0  0  0  0  0
A      0  0  0  0  0  1  1  1
G      0  1  1  1  1  1  1  1
G      0  1  1  1  1  1  1  1
T      0  1  1  2  2  2  2  2
A      0  1  1  2  2  3  3  3
B      0  1  1  2  2  3  3  4

dp[6][7] = 4 → LCS = "GTAB"
```

## 복잡도

| 항목 | 값 |
|------|-----|
| 시간 복잡도 | O(m x n) |
| 공간 복잡도 | O(m x n), 최적화 시 O(min(m, n)) |
