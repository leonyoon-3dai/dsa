# Greedy Algorithm Practice (그리디 알고리즘 연습)

Python으로 작성된 그리디 알고리즘 예제 10개. 각 파일에 한국어 라인별 주석 포함.

## Examples

| # | File | Topic |
|---|------|-------|
| 1 | `01_activity_selection.py` | Activity Selection (활동 선택) |
| 2 | `02_fractional_knapsack.py` | Fractional Knapsack (분할 가능 배낭) |
| 3 | `03_huffman_coding.py` | Huffman Coding (허프만 코딩) |
| 4 | `04_job_sequencing.py` | Job Sequencing with Deadlines (작업 스케줄링) |
| 5 | `05_minimum_platforms.py` | Minimum Platforms (최소 플랫폼) |
| 6 | `06_jump_game.py` | Jump Game (점프 게임) |
| 7 | `07_gas_station.py` | Gas Station (주유소 문제) |
| 8 | `08_interval_scheduling.py` | Interval Scheduling / Merge Intervals (구간 스케줄링) |
| 9 | `09_assign_cookies.py` | Assign Cookies (쿠키 배분) |
| 10 | `10_task_scheduler.py` | Task Scheduler (작업 스케줄러) |

## Usage

```bash
python3 01_activity_selection.py
```
